# PolitecnicoOpenWorldFlutter
Politécnico Open World (POW) es una aplicación interactiva de exploración 2D con vista *top-down* para iOS. El proyecto integra mapas del mundo real mediante OpenStreetMap (OSM) con un sistema propio de renderizado para interiores, permitiendo la transición entre exteriores y el interior de edificios, plazas y salones de clase.
